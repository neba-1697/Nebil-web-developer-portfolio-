About Me

Discover my journey as a self-taught web developer from Ethiopia

My Journey

My journey into web development began in 2023 with nothing but curiosity and a laptop. As a self-taught developer from Ethiopia, I've faced unique challenges but have turned them into opportunities for growth.

Learning to code without formal education or expensive courses meant relying on free resources, online documentation, and countless hours of practice. Every error message was a lesson, every working feature was a victory.

The lack of a strong tech community nearby pushed me to become more resourceful and independent. I learned to debug issues on my own, research solutions thoroughly, and build projects that truly tested my understanding.

Challenges That Shaped Me

  • Limited internet connectivity requiring efficient learning strategies
  • No local mentors, making me rely on online communities and documentation
  • Financial constraints that taught me to maximize free resources
  • Language barriers that improved my technical English skills
  • Time zone differences that made me more self-reliant

Daily Learning Routine

Every day, I dedicate time to learning something new. Whether it's:

  • Reading documentation for new technologies
  • Building small projects to test new concepts
  • Studying other developers' code on GitHub
  • Practicing problem-solving on coding platforms
  • Staying updated with web development trends
